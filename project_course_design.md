# 移动通信课程设计 — VHF 无线通信系统

## 基本任务

根据《2026 LX 移动通信_实验4_拓展创新 v2.pdf》第2组要求，设计超短波(VHF)无线通信传输系统。

## 系统条件（第2组 — VHF）

| 参数 | 值 |
|------|-----|
| 信号带宽 | 1 MHz |
| 载频 | 100 MHz |
| 晶振稳定度 | 0.1 ppm |
| 信道条件 | 军用车载移动信道 |
| 移动速度 | 60 km/h |
| 多径（5径） | 相对时延 [0, 0.2, 0.5, 1.0, 2.0] μs，相对功率 [0, -3, -6, -9, -12] dB |

## 系统基本组成框图（PDF 第2页）

```
发射端：
信源 → CRC → 信道编码 → 交织 → 符号调制 → 插入导频/成帧 → 上变频 → [信道]

接收端：
[信道] → 下变频 → 时间同步 → 频率同步 → 数据分离 → 信道估计 → 符号检测 → 解交织 → 译码 → CRC → 输出
```

## 总体设计要求

- 自行设计帧结构、调制方式、编码方式
- 总信源数据量 ≥ 1×10⁶ bit 的链路级仿真
- 分析和测试：时/频同步性能、信道估计性能、BER、信息速率、频谱效率
- 4个技术方向均需涉及：时间同步、频率同步、信道估计和插值、N×M MIMO均衡检测
- 核心参数需明确论证：子载波间隔、CP长度、符号周期、帧长、导频图案、多天线数量、调制阶数
- 封装 GUI 界面
